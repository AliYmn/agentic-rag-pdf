# Documentation

This directory contains the project documentation for the Agentic RAG over PDF prototype.

## Reading guide

If you are new to the repository, the most efficient order is:

1. Start with the root [README](../README.md) for setup, usage, and development commands.
2. Continue with [Architecture & Design Rationale](./architecture.md) for the system design and trade-offs.
3. Review [CLI Demo](./demo.md) for representative runs and screenshots.
4. Open [Evaluation Transcript](./demo_output.md) for the generated benchmark output.

## Document map

| Document | Purpose |
|---|---|
| [README](../README.md) | Project overview, quick start, CLI usage, tests, repository structure |
